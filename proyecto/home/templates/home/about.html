¡Hola! Mi nombre es Luis Martin Castro, tengo 25 años y soy de Buenos Aires, Argentina.

Este proyecto web fue desarrollado utilizando Python en Visual Studio Code junto con Django y Bootstrap 5.

Es mi primera experiencia en el desarrollo de una aplicación web de esta magnitud, y sé que aún tengo mucho por aprender (¡como un poco de HTML y JS!)

La idea principal del proyecto es crear un foro donde los usuarios puedan registrar sus datos y subir imágenes de animales para ser mostradas en una galería principal.

En el futuro, me gustaría expandir el proyecto agregando un blog donde los usuarios puedan comentar las imágenes.

Agradezco al profesor y tutores de Coder por su apoyo, ya que este proyecto representa un gran avance en mi camino como programador.
